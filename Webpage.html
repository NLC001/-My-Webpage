Projects

Docker-Project

Description: A PHP-based web application for managing users and administrators. The application includes user authentication, and CRUD functionalities for both users and administrators. It is containerized using Docker, making it easy to deploy across different environments.

Technologies Used: PHP, HTML, CSS, MySQL, Docker

Key Features:

  • User Authentication: Users can log in with their username and password. Admins have access to additional functionalities.
  • User Management: Admins can view a list of registered users and perform CRUD operations on users (Create, Read, Update, Delete).

Prerequisites:

Before you begin, ensure you have Docker installed on your local machine.

Installation:

  1. Clone the repository: git clone https://github.com/NLC001/Docker-Project or download the file.
  2. Build the Docker Image:
    • Navigate to the repository: cd your-repo
    • Build the image: docker build -t your-image-name .
  3. Configure the Database:
    • Import the database schema from the provided SQL file (computers.sql).
  4. Start the Docker Container:
    • Run the container: docker run -p 8080:80 -d --name your-container-name your-image-name (replace with any port of your choice)

Usage:

  1. Open the browser and navigate to http://localhost:5555/login.php.
  2. Log in with your username and password.
  3. Admins will be redirected to index.php where they can manage entries.
  4. Users will have access to their respective dashboard.
